## Usage

```bash
# installation
$ git clone https://github.com/jattoabdul/limehome-hotel-locator-fe.git
$ cd limehome-hotel-locator-fe
$ yarn (or `npm install`)

# development mode
$ yarn dev (or `npm run dev`)

# production mode
$ yarn build (or `npm run build`)
$ yarn start (or `npm start`)

# to run tests
$ yarn test (or `npm run test`)
```
